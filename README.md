# NEURA: Frontend Interface

This is the React-based frontend for the NEURA AI suite. It implements on-device neural processing via Web Workers.

## 🚀 Development Mode
1. Ensure the parent backend is running (optional, but required for history saving).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start Vite dev server:
   ```bash
   npm run dev
   ```

## 🏗️ Build for Production
```bash
npm run build
```
The output will be in `/dist`.

## 🧠 Technology
- **Transformers.js**: Core AI engine.
- **Tesseract.js**: OCR engine.
- **Tailwind CSS**: Premium design system.
